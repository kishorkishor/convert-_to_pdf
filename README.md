## How to run it ğŸš€

If you want to run this locally on your machine, it's super simple.

1.  **Clone the repo** (or just download the files).
2.  **Install dependencies**:
    ```bash
    npm install
    ```
3.  **Fire it up**:
    ```bash
    npm run dev
    ```
4.  Open `http://localhost:3000` (or `3001` if 3000 is busy) in your browser.
